## Work in progress alkoscraper

to start server:
```
npm start
```

to find price of jaloviina 0.5l one star:
```
http://localhost:3000/
```

to find specific product id: 
```
http://localhost:3000/?id=PRODUCTID
```


For example to find price of karhu sixpack:  
```
http://localhost:3000/?id=710578
```
